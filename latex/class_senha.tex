\doxysubsection{Detailed Description}
Padrao ultilizado para a senha que o usuario utilizara para acessar o sistema.

Regras do formato\+:


\begin{DoxyItemize}
\item \doxylink{class_senha}{Senha} deve conter o formato \uline{{\bfseries{XXXXX}}} onde\+:
\begin{DoxyItemize}
\item X dev ser letra maiuscula (A\+\_\+Z), minuscula(a\+\_\+z), digito(0\+\_\+9) ou sinal de pontuacao(. , ; ! ?)
\item Nao pode ter caracter duplicado.
\item Deve ter pelo menos 1 letra maiuscula.
\item Deve ter pelo menos 1 letra minuscula.
\item Deve ter pelo menos 1 digito.
\item Deve ter pelo menos 1 sinal de pontuacao 
\end{DoxyItemize}
\end{DoxyItemize}
