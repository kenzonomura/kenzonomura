\doxysubsection{Detailed Description}
Padrao ultilizado para os emails que os usuarios vao utilizar para entrar no sistema.

Regras do formato\+:


\begin{DoxyItemize}
\item \doxylink{class_email}{Email} deve seguir o formato {\bfseries{{\itshape nome}\texorpdfstring{$<$}{<}i\texorpdfstring{$>$}{>}dominio}} onde\+:
\begin{DoxyItemize}
\item Cada caracter deve ser letra maiuscula, minuscula, digito ou ponto(.).
\item Nao podem ter pontos em sequencia.
\item nome deve conter de 2 a 10 caracteeres, e o ultimo caracter nao pode ser ponto(.).
\item dominio deve conter de 2 a 20 caracteres, e o primeiro caracter nao pode ser ponto(.). 
\end{DoxyItemize}
\end{DoxyItemize}
